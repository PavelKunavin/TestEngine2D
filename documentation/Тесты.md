// Создание сцены вместо ручного создания тел:
g_scene = te2d_scene::create();

Этап 2D.3 — Анимация + Аллокатор

Файл: app2d/main.cpp (дополнение к 2D.2)

Обновления под аллокатор:

```cpp
// Аллокатор для анимаций:
te2d_allocator g_anim_allocator;

// в game_init():
g_anim_allocator = te2d_allocator::create(1024 * 64); // 64 KB

// Тестовая анимация (заглушка — один кадр = полная текстура):
te2d_anim_frame idle_frames[] = {
    { {0, 0, (f32)tex_w, (f32)tex_h}, 0.1f } // один кадр
};
te2d_animation g_anim_idle = te2d_animation::create(g_anim_allocator, "idle", idle_frames, 1, true);

// Аниматор:
te2d_animator g_player_animator;
g_player_animator = te2d_animator::create(&g_player_sprite);
g_player_animator.play(&g_anim_idle);

// в game_update():
g_player_animator.update(dt);

// в game_shutdown():
g_anim_allocator.destroy();
```

Что проверять:

# Проверка Ожидаемый результат Если баг
1 Спрайт не дёргается Один кадр — статично таймер/UV сбой
2 ImGui Текущий кадр, таймер, имя анимации —
3 Батч и мир на аллокаторах Сборка без ошибок не обновлён create()
